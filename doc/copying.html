The ColdFrame runtime is licensed under the GNAT-modified GPL or GMGPL. This adds to the standard GPL statement 

   --  As a special exception, if other files instantiate generics from
   --  this unit, or you link this unit with other files to produce an
   --  executable, this unit does not by itself cause the resulting
   --  executable to be covered by the GNU General Public License.  This
   --  exception does not however invalidate any other reasons why the
   --  executable file might be covered by the GNU Public License.

The XSLT stylesheets are licensed under the GPL with the following addition, which has been adapted from the GNU Bison template's statement: 

   As a special exception, when portions of this file are copied by
   a stylesheet processor into an output file, this file does not by
   itself cause the resulting file to be covered by the GNU General
   Public License.  This exception does not however invalidate any
   other reasons why the output file might be covered by the GNU
   Public License.
