ColdFrame: Scripting the Event Queue

ColdFrame.Scripted_Testing_G
Provides the procedure Register to tell the framework
  • which Dispatcher (event queue) is in use
  • and the Initialize procedure to call to initialize all the domains,
and the commands initialize, start_dispatcher and wait_until_idle.
To use wait_until_idle, the event queue in use must be a Test or Test_Trace queue.
ColdFrame.Project.Scripted_Testing
Instantiates ColdFrame.Scripted_Testing_G with the project Event Queue.
