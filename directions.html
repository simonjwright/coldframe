ColdFrame: Directions

Short-term

  • Improve unit test support:
    • Change «singleton» code generation so that Tear_Down and re-Initialize can reset the instance.
    • Provide an event queue that test procedures can block on until there are no more events.
    • Tear_Down the event queue (otherwise you have to take drastic measures to end the test harness).

Longer-term

  • provide model validation support.
  • provide at least one alternate architecture.
